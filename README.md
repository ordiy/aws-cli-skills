## 🚀 Installation

### Option 1: Direct Install via ClawHub (Recommended)
```bash
npx clawhub install watadot-aws-s3
```

### Option 2: Clone into Workspace
Clone the repo into your OpenClaw workspace `skills/` directory:
```bash
cd ~/.openclaw/workspace/skills
git clone https://github.com/ordiy/aws-cli-skills.git
```
